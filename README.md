# skills

Personal agent skills distributed through the [Skills Marketplace](https://skills.sh/).

List the skills in this repository:

```sh
npx skills add youngxguo/skills --list
```

Install every skill globally for Codex and Claude Code:

```sh
npx skills add youngxguo/skills --global --agent codex claude-code --skill '*'
```

Replace `'*'` with a skill name to install only that skill.
